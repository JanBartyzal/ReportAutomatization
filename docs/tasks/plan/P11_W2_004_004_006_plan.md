## Implementation Notes

### Test Patterns
1. Use `@SpringBootTest` for integration tests where needed
2. Use `@WebMvcTest` for controller tests with `@MockBean`
3. Use `@DataJpaTest` for repository tests
4. Follow AAA pattern (Arrange, Act, Assert)
5. Include both positive and negative test cases

### Migration Considerations
1. Handle CASCADE deletes properly
2. Consider data loss warnings in comments
3. Maintain proper ordering (U-migrations run in reverse order of V-migrations)
4. Handle seed data deletions carefully
