# CIM — Complete Vector Page Wireframes

This directory contains the 100% vector-based page layouts for **CloudInfraMap (CIM)**.

## Key Files

1.  **[pages_wireframes_complete.svg](pages_wireframes_complete.svg)**:
    - **All 12 Pages**: Vector wireframes for Dashboard, Compliance, Designer, Admin, Pricing, etc.
    - **Responsive**: Includes specific Tablet (768px) and Mobile (375px) layouts.
    - **Localization**: Unified branding and CZK (Kč) currency throughout.

2.  **[special_overlays.svg](special_overlays.svg)**:
    - **Feature Locking**: 3 modes of locked feature previews (Blur, Skeleton, Teaser).
    - **System States**: Error states (404, 500) and Offline mode.

## Implementation Notes

- **Grid**: All desktop wireframes follow a 12-column grid within a 1440px viewport.
- **Components**: These wireframes use the vector elements defined in `../Components/`.
- **States**: Every page layout is provided as a editable source for both Light and Dark modes.
